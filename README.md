# 🔔 Arduino IR Sensor Burglar Alarm System

## 📌 Project Overview
This project is a simple **Burglar Alarm System** built using an **Arduino Uno**, an **IR sensor**, a **buzzer**, and an **LED**.

The system detects the presence of an object using the IR sensor. When an object is detected:
- The **buzzer sounds**
- The **LED blinks**

This project is suitable for beginners learning Arduino and basic embedded systems.

---

## 🛠 Components Used

- Arduino Uno
- IR Obstacle Detection Sensor
- Buzzer
- LED
- 220Ω Resistor
- Breadboard
- Jumper Wires
- USB Cable

---

## 🔌 Circuit Connections

### IR Sensor
- VCC → 5V (Arduino)
- GND → GND (Arduino)
- OUT → Digital Pin 13

### Buzzer
- Positive (+) → Digital Pin 9
- Negative (–) → GND

### LED
- Anode (+) → Digital Pin 11 (through 220Ω resistor)
- Cathode (–) → GND
